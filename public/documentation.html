myFlix API

This is a REST API for the application called myFlix that intereacts witha database used to story data about movies. Users will be able to create a new user, make changes to their account such as add/remove username, change username,edit their favorite movies and also delete account. Users will be able to access data about movies. The data that can be accessed includes, Title, director, genre, description, genre data and information abou the director. This api will be able to be used to access data using the get, post, delete and delete methods. this API is created using Node.js, javascript and express.
